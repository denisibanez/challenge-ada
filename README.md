# Desenvolvimento de um Quadro de Kanban

# Informações
* Esse projeto usa:
* Eslint
* Angular
* Reative Forms
* Angular Material
* Angular Router
* @ngxs/store
* Angular Imask
* Date Fns
* ng2-currency-mask
* husky
* Prettier
* jest
* Typescript
